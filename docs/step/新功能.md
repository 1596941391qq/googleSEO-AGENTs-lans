第一部分：精简的 AI 代理系统（4 个核心代理）
SEO 研究官：分析搜索引擎偏好、获取竞争对手结构、提取关键词和 LSI 关键词、生成 GEO/AIO 优化建议
内容写手：基于 SEO 建议生成初稿、遵循竞争对手结构、注入关键词、优化 GEO/AIO
质量把关官：检查关键词密度、检测 AI 痕迹、验证 GEO/AIO 合规性、生成质量评分
图文创意官：提取主题、生成 Nano Banana 2 Prompt、调用 API、集成图像
第二部分：完整的内容创作链路（8 个步骤）
搜索引擎偏好分析：Google、ChatGPT、Claude、Perplexity 的不同偏好
竞争对手分析：获取 Top 10 竞争对手的结构、内容框架、内容缺口
关键词优化：关键词密度、LSI 关键词、最佳位置规划
GEO 优化：地理位置相关内容、本地案例研究、地区特定数据、本地化措辞
AIO 优化：Q&A 格式、答案块结构化、AI 友好的措辞、引用优化、结构化数据
内容生成：基于所有优化建议生成最终文章
质量检查：关键词密度、AI 概率、可读性、GEO/AIO 合规性
图像生成：4-6 张 Nano Banana 2 生成的高质量图像
第三部分：用户平台工作流程
第 1 阶段：挖词：输入网站地址或初始关键词 → 系统识别蓝海词 → 用户选择目标关键词
第 2 阶段：生成文章：4 个代理协作 → 实时进度显示 → 用户预览和编辑
第 3 阶段：发布：选择平台 → 配置参数 → 发布 → 获取链接
第 4 阶段：追踪（可选）：监测排名、追踪流量、检查 AI 引擎引用，调用 google console search
第四部分：Nano Banana 2 集成
模型特点：快速（5-10s）、高质量（4K）、成本低（$0.05-0.08/image）
Prompt 生成策略：自动提取 4-6 个主题 → 为每个主题生成优化的 Prompt
API 集成：完整的请求/响应格式示例
图像处理：下载、元数据添加、位置规划
